# **Frontend Coding Challenge: Product Management Dashboard**

### **Objective**

Build a React application that allows users to browse, search, and manage a collection of products. This task evaluates your ability to handle data fetching, state management, and building a functional UI using modern React patterns.

### **Timeframe**

Please complete and submit this within **48 hours**.

---

### **1. Technical Stack**

- **Framework:** React
- **Language:** TypeScript
- **Styling:** Any CSS framework or library of your choice (Tailwind, SCSS, etc.)

### **2. Data Layer Requirements**

You are required to set up your own data source for this project to ensure data consistency. You may choose one of the following two options:

- **Option A:** Use [mockapi.io](https://mockapi.io/) to create a product resource and fetch/update data from there.
- **Option B:** Use a local `JSON` file within your project to simulate a mock API environment.

### **3. Core Features**

#### **A. Product Display & Discovery**

- Fetch and display a list of products in a responsive layout.
- Each product should show a title, image, price, and category.
- Implement a **Search Bar** to filter products by name.
- Implement a **Category Filter** to narrow down the list (e.g., Electronics, Fashion, etc.).

#### **B. Selection & Management**

- Users should be able to "Favorite" or "Select" specific products.
- **Persistence:** Your application must ensure that the user's selected/favorite items remain saved even after the browser page is refreshed.

#### **C. User Experience**

- Handle the "Loading" state while data is being fetched.
- Handle "Empty" states (e.g., when a search returns no results).
- Ensure the layout is responsive for both desktop and mobile screens.

---

### **4. Submission Instructions**

1. **Repository:** Host your code on GitHub or GitLab with a clear, descriptive commit history.
2. **README:** Include a brief document explaining:

- How to run the project locally.
- A high-level overview of your technical choices and architectural patterns.
